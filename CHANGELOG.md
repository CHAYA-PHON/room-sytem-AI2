# 📝 ประวัติการปรับปรุงรุ่น (System Changelog) - Sabaidee Dormitory

เอกสารฉบับนี้ใช้สำหรับบันทึกประวัติการปรับปรุงระบบ พัฒนาคุณสมบัติใหม่ และแก้ไขปัญหาต่าง ๆ ในแต่ละรุ่นของแอปพลิเคชัน **Sabaidee Dormitory** เพื่อให้ผู้ดูแลระบบและนักพัฒนาระบบในอนาคตสามารถตรวจสอบและอ้างอิงได้อย่างถูกต้องและแม่นยำ

## 💡 แนวทางการบันทึกเพิ่มเติมในอนาคต
1. เมื่อมีการส่งโค้ดไปยัง GitHub หรือเผยแพร่เวอร์ชันใหม่ (Publish) ให้เพิ่มเลขรอบ `PBXX` (เช่น `PB02` ➡️ `PB03`)
2. หากมีการเพิ่มฟีเจอร์หรือความสามารถใหม่ ให้ปรับเพิ่มเลขหลักแรกของรุ่นย่อย `M` (เช่น `_1.3` ➡️ `_2.0`)
3. หากเป็นการแก้ไขบั๊กหรือปัญหาทางเทคนิค ให้ปรับเพิ่มเลขตัวหลังของรุ่นย่อย `B` (เช่น `_1.3` ➡️ `_1.4`)
