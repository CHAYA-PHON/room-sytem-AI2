# 📘 คู่มือการติดตั้ง ระบบจัดการหอพัก (Google Apps Script + Google Sheets)

ระบบจัดการหอพักนี้ถูกพัฒนาขึ้นโดยใช้ **Google Apps Script** เป็น Backend/API และใช้ **Google Sheets** เป็นฐานข้อมูลจัดเก็บข้อมูลระบบทั้งหมด 8 ตารางหลัก โดยไม่มีการใช้งานเซิร์ฟเวอร์ภายนอก ช่วยรักษาความปลอดภัยและความปลอดภัยของข้อมูลได้ 100%

---

## 🧱 โครงสร้างฐานข้อมูลสเปรดชีต (8 ชีตหลัก)
เมื่อเรียกใช้งานแอปพลิเคชันเป็นครั้งแรก ระบบจะทำการสร้างชีตและหัวตารางเหล่านี้ให้โดยอัตโนมัติ:
1. **`ห้อง`** - เก็บรหัสห้อง ชื่อห้อง ค่าเช่า ค่าน้ำ-ไฟขั้นต่ำ วิธีชำระเงินหลัก หมายเลขบัญชี และรายละเอียดหมายเหตุ
2. **`ผู้เช่า`** - เก็บข้อมูลผู้เช่าผูกกับห้องพัก เบอร์โทร วันเข้าเริ่มทำสัญญา และเลขมิเตอร์ตั้งต้นน้ำ-ไฟ
3. **`มิเตอร์`** - บันทึกเลขมิเตอร์น้ำ-ไฟประจำแต่ละเดือน พร้อมบันทึกหมายเหตุการแก้ไขค่า
4. **`รายการเพิ่ม`** - รายการค่าใช้จ่ายบริการเสริมรายห้องพักในแต่ละเดือน (เช่น ค่าย้ายของ ค่าอินเทอร์เน็ต)
5. **`บิล`** - สรุปรายละเอียดบิลคำนวณอัตโนมัติ ยอดสุทธิ และยอดค้างจ่าย
6. **`บัญชี`** - เก็บธนาคาร บัญชีรับฝากเงินเพื่อนำไปโชว์ท้ายบิลแจ้งหนี้
7. **`ประวัติชำระ`** - บันทึกประวัติการนำโอนเงินและตัดชำระยอดบิลค้างสะสมด้วยหลัก FIFO
8. **`ADMIN`** - เก็บบัญชี Username และ PIN 4 หลัก สำหรับแอดมินล็อกอินระบบหน้าบ้าน

## 🚀 ขั้นตอนการติดตั้งและ Deploy เพื่อใช้งาน

### ขั้นตอนที่ 1: สร้าง Google Sheets ใหม่
1. ไปที่ [Google Sheets](https://sheets.google.com) และกดสร้าง **สเปรดชีตเปล่า** ขึ้นมาใหม่
2. คัดลอก **ID ของสเปรดชีต** จากช่อง URL (จะอยู่ในรูป `https://docs.google.com/spreadsheets/d/YOUR_SPREADSHEET_ID/edit`)

### ขั้นตอนที่ 2: ตั้งค่า Google Apps Script
1. ที่แถบเมนูด้านบนของสเปรดชีต ไปที่ **ส่วนขยาย (Extensions)** -> **Apps Script**
2. ลบโค้ดเดิมทั้งหมดในไฟล์ `รหัส.gs` (หรือ `Code.gs`) และวางโค้ดจากไฟล์ `docs/apps-script/Code.gs` ลงไป
3. กดปุ่ม `+` ข้างๆ ตัวเลือกไฟล์ในบาร์ซ้ายเพื่อเพิ่มไฟล์ **HTML** ตั้งชื่อตามไฟล์ดังต่อไปนี้แล้วนำโค้ดไปวาง:
   - **`Index.html`** (เอาโค้ดจาก `docs/apps-script/Index.html` ไปใส่)
   - **`Styles.html`** (เอาโค้ดจาก `docs/apps-script/Styles.html` ไปใส่)
   - **`App.html`** (เอาโค้ดจาก `docs/apps-script/App.html` ไปใส่)

### ขั้นตอนที่ 3: ตั้งค่า Script Properties
1. ที่แถบซ้ายของ Apps Script ให้คลิกที่ไอคอน **การตั้งค่าโครงการ (Project Settings)** (รูปฟันเฟือง)
2. เลื่อนลงมาด้านล่างสุดที่หัวข้อ **คุณสมบัติสคริปต์ (Script Properties)** และกด **เพิ่มคุณสมบัติสคริปต์ (Add script property)**
3. ตั้งค่าคีย์และค่าดังต่อไปนี้:
   - **คุณสมบัติ (Property)**: `SPREADSHEET_ID`
   - **ค่า (Value)**: `วาง ID สเปรดชีตที่คัดลอกมาจากขั้นตอนที่ 1`
4. กดบันทึกการตั้งค่าคุณสมบัติสคริปต์

### ขั้นตอนที่ 4: การเผยแพร่เว็บแอป (Deploy Web App)
1. คลิกที่ปุ่ม **การทำให้ใช้งานได้ (Deploy)** บริเวณมุมบนขวา -> เลือก **การจัดการทำให้ใช้งานได้ใหม่ (New deployment)**
2. คลิกเลือกประเภทเป็น **เว็บแอป (Web app)** (ไอคอนรูปฟันเฟือง)
3. กำหนดรายละเอียดการตั้งค่าดังนี้:
   - **คำอธิบาย (Description)**: `ระบบจัดการหอพัก v1.0`
   - **เรียกใช้เป็น (Execute as)**: เลือก **ฉัน (Me / บัญชีอีเมลของคุณ)**
   - **ผู้มีสิทธิ์เข้าถึง (Who has access)**: เลือก **ทุกคน (Anyone)**
4. คลิกปุ่ม **การทำให้ใช้งานได้ (Deploy)**
5. ระบบจะขออนุญาตสิทธิ์เข้าถึงพื้นที่ชีต (Authorize access) ให้ทำตามขั้นตอนการยืนยันตัวตนจนเสร็จสิ้น
6. คัดลอก URL ของเว็บแอปที่ปรากฏขึ้น เพื่อใช้แชร์และนำไปเปิดเข้าใช้งานระบบจัดการหอพักของคุณได้ทันที!
